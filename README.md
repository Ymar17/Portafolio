# Portafolio

Este es mi portafolio personal, un sitio web simple que muestra mis habilidades, proyectos y experiencia profesional.

## Características

*   **Diseño responsivo:** Se adapta a diferentes tamaños de pantalla (escritorio, tablet, móvil).
*   **Navegación intuitiva:** Menú de navegación claro y fácil de usar.
*   **Secciones básicas:** Presentación, proyectos, habilidades, experiencia y contacto.
*   **Código limpio:** Estructura organizada en HTML, CSS y JavaScript puro.

## Requisitos previos

Para ver este proyecto solo necesitas:

*   Un **navegador web** moderno (Chrome, Firefox, Edge, etc.).

## Instalación y configuración

Puedes ver el portafolio directamente abriendo el archivo `index.html` en tu navegador.

Si deseas tener una copia local:

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/Ymar17/Portafolio.git
    ```

2.  **Accede al directorio del proyecto:**
    ```bash
    cd Portafolio
    ```

3.  **Abre el archivo principal en tu navegador.**

## Estructura del proyecto

```
Portafolio/
├── index.html          # Archivo principal con la estructura HTML
├── css/
│   └── style.css      # Todos los estilos del sitio
├── js/
│   └── script.js      # Funcionalidad interactiva (ej. menú móvil)
└── README.md
```

## Uso

Este es un sitio web estático. Simplemente abre `index.html` en tu navegador para ver el portafolio.
